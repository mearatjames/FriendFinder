# FriendFinder

## Express JS homework
- Using Node JS and Express JS for app functionality

### How the app works

1. User is greeted with a simple landing page with a button that is linked to the survey page.
2. The survey will ask for Name, profile photo URL.
3. User will also have to answer all 10 questions to find the closest match based on the answer given by the user.
4. Once the best match is found, a modal with the name and photo of that match.
